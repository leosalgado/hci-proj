A relação entre a pobreza e o trabalho infantil

A sociedade sofre de muitos problemas mundiais, sendo um deles o trabalho infantil, esse que vem piorando a cada dia. Muitas crianças são forçadas a trabalhar para ajudar suas famílias em seus respectivos lares. Em 2020, mais de cem milhões de crianças foram vítimas do trabalho infantil, ou seja, aproximadamente uma em cada dez crianças acaba sofrendo com estes tipos de problema. Assim como diz Luciano Nakabashi (2017): “A necessidade de uma família pobre aumentar a renda por meio de outros membros do grupo familiar, inclusive das crianças, é que faz a relação entre a pobreza e o trabalho infantil”, assim o objetivo desta pesquisa foi analisar e relacionar os dois problemas, para que assim, fosse possível chegar em uma solução adequada para resolver essa situação.

No trabalho serão usados dois Datasets abertos diferentes, sendo um deles disponibilizado pelos dados abertos do Governo Federal. Já no segundo Dataset, mostra informações sobre a erradicação do trabalho infantil, o mesmo divulgado pelo portal da transparência. A linguagem escolhida foi Python, por ser de alto nível e popular no mercado, com várias bibliotecas e tecnologias desenvolvidas para ela. A biblioteca escolhida é a Pandas, por ser a principal ferramenta para a manipulação e limpeza de dados. Foi utilizado no código, os datasets para criação de gráficos, a fim de melhorar a análise das situações. Com isso, foi utilizada a biblioteca matplotlib.pyplot para a criação dos gráficos que puderam ser usado pra estudar o assunto.

Para a avaliação do site utilizado para a obtenção do dataset sobre pobreza, foi decidido a utilização do método de avaliação por inspeção, mais especificamente a avaliação heurística. Durante a avaliação algumas falhas foram encontradas nos dois sites, no primerio foi encontrado falhas como: um botão de relacionados, onde o usuário clica e não acontece nada, um botão "ações" onde sua única função é aparecer outros dois botões, entre outros. No segundo site, as falhas encontradas foram: ao clicar no botão "acessibilidade" não acontece nada e nem aparece nenhuma mensagem de erro, uma má disposição da aba de dados abertos, entre outros. Foi feita uma avaliação de acessibilidade no site ASES do governo federal para os dois sites, os resultados foram: 73,43% para o primeiro, sendo a maior parte dos erros técnicos e 99.19% para o segundo, uma pontuação quase perfeita.

Para uma melhor compreensão e análise dos dados que foram retirados dos sites e organizados pela pesquisa, foi criada uma interface interativa, onde ela foi feita inteiramente em python, usando as bibliotecas streamlit para a criação de toda a interface onde será exibido os gráficos e informações e a biblioteca pyplot para transformar os dados em gráficos, para uma melhor visualização. Para que consiga ser feita uma avaliação minuciosa dos dados, foram filtrado os gráficos por meses do ano de 2021, portanto, na interface possui uma aba onde o usuário pode selecionar o mês em que deseja analisar e a partir disso ele verá os gráficos apenas desse mês, facilitando a compreensão.

Após a análise dos dados e dos gráficos que foram criados, é possível perceber que o valor da parcela recebido do auxílio do trabalho infantil tem pouco impacto em pessoas e famílias em situação de extrema pobreza e um impacto mediano em pessoas e famílias em situação de pobreza, portanto nossa premissa inicial não pode ser confirmada com base apenas nestes dados. Com a ferramenta interativa que foi criada, é possível set estudado mais profundamente esses dados, assim cada um pode tirar suas conclusões sobre o assunto, ela foi feita para que todas as pessoas consigam usá-las, tais como: estudantes, jornalistas ou apenas um cidadão comum que não apresenta altos conhecimentos de programação e está em busca de dados sobre o assunto abordado.
